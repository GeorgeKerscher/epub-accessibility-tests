history-010 Four single follow link and return tests

This test involves four parts. It tests going to a footnote, and then returning, and the same process for an end note, a glossary entry, and a bibliographic reference.

Steps

  1. Follow the link to the footnote. 1.
  2. Return here using the go back function.
  3. Follow the link to the end note. 2.
  4. Return using the go back function.
  5. Follow the link to the glossary reference. database.
  6. Return to this location using the go back function.
  7. Follow the link to the bibliographic reference. A Collection of Important Writings.
  8. Return to this location using the go back function.

If all four of the links through the book are traversed and the reading system's go back function returns to this location each time, the test passes. Not any issues encountered.

Indicate Pass or Fail.

history-020 Return after a search

This test determines if the reading system's search function supports the go back function. Ideally, the go back function avoids taking the user to the search box, i.e. you do the search and return directly to this location.

Steps

  1. Search for "Apollo 13" and go to the chapter containing that information.
  2. Using the go back function, return to this location. Note if the go back function took you directly back to this location or if it to you to the search dialogue.

The test passes if you can go directly back to this location from the chapter containing Apollo 13.

Indicate Pass or Fail.

history-030 Return after go to page.

This test determines if the Go to page function supports go back functionality.

Steps

  1. Use the go to page function in the reading system and navigate to page 20.
  2. After reading some text, use the go back function to return to this location. Note if the go back function took you to the go to page dialogue before returning to this location.

The test passes if you are returned to this location without going through the go to page function.

Indicate Pass or Fail.

history-040 Using nav TOC to move and return

This test determines if the reading system built in TOC, i.e., using the nav doc is excluded from the history list.

Steps

  1. Use the reading systems utility to review the items in the nav doc. Note this is not the short TOC that is listed in the spine. Select the "sample Text" chapter and go there and read some content.
  2. Use the go back function to return to this location. Note if the go back function takes you to the nav doc or if if it returns you directly to this location.

The test passes if the go back function avoids the nav doc display.

Indicate Pass or Fail.

history-050 Multiple items in the history

This test identifies how many items can be stored in the history list.

Steps

  1. Use the nav document to go to the short TOC.
  2. Then scroll down to the index and go there.
  3. Then select the link to front matter and activate the link.
  4. Then use the go back function to return to the index and activate the link to the introduction.
  5. Finally, use the go back function to get back to this location. Note how many go back steps the reading system supports.

The test passes if you get back to this location. The history should keep track of all jumps to different locations. Only the TOC from the nav doc should be not included.
